# DrainioMart - Smart Drainage IoT Website

Website SmartWater & Smart Drainage berbasis HTML, CSS, dan JavaScript.

## Fitur
- Responsive Mobile
- Tombol WhatsApp langsung
- Sistem pembayaran popup
- Pilihan pengiriman JNE & J&T
- Desain modern
- Katalog produk IoT

## Cara Upload ke GitHub Pages

1. Upload semua file ke repository GitHub
2. Masuk ke:
   Settings → Pages
3. Source:
   Deploy from branch
4. Branch:
   main
5. Save

Website otomatis online.
